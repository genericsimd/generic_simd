Detailed Description

SIMD LANES=4 interfaces implemented by scalar The source file is organized as follows I. all types definition, with class function incorporated II. data operation interfaces 1. load/store 2. select 3. broadcast/rotate/shuffle/smear/setzero 4. gather/scatter 5. load const, smear const, load and splat 6. masked load/masked store III. Mask type (i1) interfaces 1. mask construction 2. bit operations IV. General data operation interfaces 1. Unary 2. Math unary 3. Binary 4. Ternary 5. Compare 6. Max/Min 7. Reduce 8. Cast.
